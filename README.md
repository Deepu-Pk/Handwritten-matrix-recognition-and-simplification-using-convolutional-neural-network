# Mini-Project
HANDWRITTEN MATRIX RECOGNITION AND SIMPLIFICATION USING CONVOLUTIONAL NEURAL NETWORK.
An AI system for doing matrix operations by taking an input image of the handwritten matrix<br/>
Python Library : Tensorflow-Keras,Numpy,Opencv and seaborn
<h4>Model.py<h4/><br/>
  1. CNN model is developed by tensorflow-keras <br/>
  2. Data set : MNIST<br/>
  <br/>
<h4>Datapreprocessing<h4/><br/>
  1. Convert image from RGB to Gray scale<br/>
  2. Convert Grayscale image into binary image<br/>
  3. segment the image and select each digit in that image<br/>
  4. Predict the number <br/>
  5. Perform the linear algebra operation<br/>
